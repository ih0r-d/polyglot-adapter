# Polyglot Platform Documentation

Welcome to the official documentation of Polyglot Platform.

This site provides architectural overview, runtime concepts, and build-time tooling documentation for the platform.

---

## Platform Overview

Polyglot Platform is structured into two independent layers:

### Runtime (JDK 25)

Responsible for:

- GraalVM context management
- Script execution
- Value mapping
- Spring Boot integration

### Tooling (JDK 21)

Responsible for:

- Contract model definition
- Deterministic interface generation
- Maven build integration
